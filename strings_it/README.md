Can you find the flag in file without running it?

[Challenge](../strings_it/strings_it)

## Solution

1. Check the file extension.<br>
   `file strings`
2. Get any usable/valuable information from the file.<br>
   `strings strings` or `cat strings`
3. Get the flag.
